# kubernetes-ha-binary

## 项目介绍
项目致力于让有意向使用原生kubernetes集群的企业或个人，可以方便的、系统的使用**二进制**的方式手工搭建kubernetes高可用集群。并且让相关的人员可以更好的理解kubernetes集群的运作机制。

## 软件版本
- os centos7.2（ubuntu也适用，需要替换部分命令）
- kubernetes v1.16.4
- etcd v3.3.18
- docker 17.09.1-ce
- calico v3.10.2
